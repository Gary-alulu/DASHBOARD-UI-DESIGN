# Unique Dashboard

A modern, feature-rich learning management dashboard built with React and Vite, featuring a responsive design and dark mode support.

## Features

- **Responsive Layout**: Fully responsive design that works seamlessly across desktop, tablet, and mobile devices
- **Dark Mode Support**: Built-in dark mode with custom color palette
- **Interactive Components**:
  - Course Progress Tracking
  - Weekly Activity Monitoring
  - Daily Schedule Management
  - New Courses Discovery
  - Community Engagement
- **Modern UI Elements**:
  - Custom Tailwind CSS styling
  - Animated glowing effects
  - Custom color schemes with pastel variants

## Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: 
  - Headless UI
  - Hero Icons
  - React Icons
- **Type Checking**: ESLint

## Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn package manager

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

### Building for Production

Create a production build:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Customization

### Theme Configuration

The dashboard uses a custom Tailwind CSS configuration with extended colors and animations. Modify `tailwind.config.js` to customize:

- Color schemes
- Font families
- Shadow effects
- Animations

## Available Routes

- `/` - Dashboard Home
- `/my-courses` - Course Management
- `/my-classes` - Class Schedule
- `/messages` - Messaging System
- `/notifications` - Notifications Center
- `/calendar` - Calendar View
- `/community` - Community Hub
- `/settings` - User Settings
